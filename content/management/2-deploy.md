# Github Pages

Naked Doc ではデプロイ先のデフォルトを**Github Pages**にしています。
以下のコマンドを実行することでリポジトリ先に gh-pages ブランチを作成して、静的ページを自動でデプロイします。

```sh
$ npm run deploy
```
